# 💻 arq1

Este repositório reúne todo o material acadêmico e laboratórios realizados na disciplina **Arquitetura de Computadores I**. O objetivo é centralizar os guias, exercícios em Verilog e arquivos de teste utilizados nas aulas e nas avaliações.

## 🧠 Objetivo

Este material serve como suporte de estudo para entender:
- Fundamentos de lógica digital e organização de computadores
- Linguagem Verilog para descrição de hardware
- Técnicas de simulação e verificação de circuitos
